##Setup Qt Environment
1. Download and install Qt from the official website: https://www.qt.io/download
2. Open Qt Creator and configure the necessary kits for your development environment.
3. Open the RoomlyChat project by navigating to the `client` directory and selecting the `RoomlyChat.pro` file.
4. Build and run the project using Qt Creator.

# RoomlyChat
A lightweight, room-based chat application built with Qt. Instant messaging without login, just join and start chatting. Invite you friends with their unique RoomlyTag to your rooms.
